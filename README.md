## 项目介绍

基于vue3框架的后台管理系统，基于新峰商城进行的仿写，主要是为了学习Vue3和Element Plus的组合式开发，主要实现的功能如下：

1、系统介绍

2、添加商品

3、首页配置

4、模块管理

5、系统管理

## 构建说明

本地调试：

```javascript
npm run dev
```

项目部署：

```javascript
npm run build
```
